Объекты Array и String

Дана web-страница (по ссылке загрузится архив с исходными файлами). HTML и CSS уже написаны, нужно написать JavaScript-код, реализующий обработку нажатия кнопок. Если нужно, внесите необходимые изменения в HTML-код страницы, например, можно добавить идентификаторы (атрибуты id) элементам управления.

Для отделения контента и поведения веб-страницы, используйте ненавязчивый Javascript (см. лекцию по DOM)

Добавьте следующее поведение для кнопок:

  • Очистить: Удаляет весь текст из текстового поля.
  • Верхний регистр: Преобразует весь текст в верхний регистр.
  • Сортировать: Сортирует строки в алфавитном порядке.
  • Перевернуть: Меняет порядок строк на обратный.
  • Добавить номера строк: Добавляет номера строк вначале каждой строки, например, так "1. " Не обращайте внимание, если строки уже пронумерованы.
  • Удалить пустые строки: Удаляет все пустые строки из текстового поля, включая строки заполненные только пробелами (используйте метод trim()).
  • Перемешать: Записать строки в случайном порядке. У JavaScript-объекта Array отсутствует метод shuffle, но вы можете реализовать следующий алгоритм для перемешивания: 
    для каждого индекса i:
   случайно выбрать индекс j >= i.
   поменять метами строки i и j.

Помните, что

  • вы можете обратиться к DOM-объекту любого HTML-элемента с помощью метода document.getElementById("идентификатор").
  • У объекта String есть метод split для разделения строки на подстроки и занесения в массив.
  • У объекта Array есть свойство length и такие методы как: concat, join, pop, push, reverse, shift, slice, sort, splice и unshift.
