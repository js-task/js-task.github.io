Объекты DOM: Цветные прямоугольники

Добавить javascript-код к странице (скачать файлы в архиве).

Используя объекты DOM, напишите программу, которая

  • при загрузке страницы создает N (число N задать глобальной переменной в js-файле) прямоугольников (элементов div с классом rectangle) и помещает их в элемент div с идентификатором rectanglearea. Прямоугольники должны располагаться на странице случайным образом.
  • при нажатии кнопки "Пуск!" каждый прямоугольник должен начать изменять свой цвет случайным образом 1 раз в секунду (если смена цветов запущена, кнопка "Пуск" не должна реагировать на повторные нажатия).
  • при нажатии кнопки "Стоп" смена цветов прямоугольников должна прекратиться.
  • при клике по созданному прямоугольнику, этот прямоугольник должен быть удален (в теле обработчика события ключевое слово this ссылается на объект, с которым произошло данное событие).
