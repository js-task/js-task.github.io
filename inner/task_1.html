Объект Date

Дана HTML-страница и ее стиль оформления (скачать одним архивом). Напишите необходимый JS-код для обработки нажатия кнопки "Выходной?"

  1. Вставить слово "ДА!" (если текущий день недели суббота или воскресенье) или "НЕТ!" в элемент с идентификатором answer.
  2. Изменить class элемента body на yes или no соответственно.
  3. Вставить текущие дату, включая название дня недели и название месяца на русском языке (можно воспользоваться массивами), а также текущее время в элемент с идентификатором now.

По ссылке можно посмотреть описание объекта Date. Воспользуйтесь методами этого объекта для получения номера дня недели.

Для обращения к телу документа используйте конструкцию document.body.

Для изменения атрибута class у элемента используйте свойство className, например так: 

var span = document.getElementById("qwerty");
span.className = "myClass";
Тогда элементу с идентификатором qwerty будет прописан класс myClass.

Для изменения содержимого HTML-элемента используйте свойство innerHTML.
