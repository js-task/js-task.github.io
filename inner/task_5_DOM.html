События таймера

Добавьте javascript-код на страницу (скачать файлы в архиве).

Напишите скрипт, который запускается по нажатию кнопки "Пуск" и отсчитывает время до 0 минут 0 секунд, а затем выдает текст напоминания, заданного пользователем, и меняет фоновый цвет страницы на цвет, заданных пользователем.

  • минуты и секунды обратного отсчета должны отображаться на странице каждую секунду в соответствующих полях для ввода (поля с идентификаторами min и sec);
  • обратный отсчет должен остановиться, когда время станет равно 0 минут 0 секунд;
  • необходимо реализовать возможность повторного запуска обратного отсчета (при этом нужно сбросить фоновый цвет страницы);
  • кнопка "Пуск" не должна реагировать на нажатие, если отсчет уже начат;
  • по нажатию на кнопку "Пауза" отсчет должен остановиться;
  • по нажатию на кнопку "Стоп" отсчет должен остановиться, значения полей сброситься в 0.
